Gerador de Números para Loteria(Política de Privacidade)

br.uware.geradormega_sena


Estamos utilizando os serviços do Firebase(firebase.google.com) para coleta de dados.
A coleta de dados do Firebase está descrita neste site.
Este aplicativo usa o Google Analytics, que é uma das soluções de análise mais difundidas e confiáveis ​​da Web, para nos ajudar a entender como você usa o aplicativo e como podemos melhorar sua experiência. Esses serviços podem rastrear itens como quanto tempo você gasta no aplicativo e as páginas visitadas, para que possamos continuar produzindo conteúdo atraente.
A coleta de dados é compartilhada com o Google Analytics

Utilizamos o recurso crashlytics do firebase que coleta dados do telefone quando o aplicativo gera uma falha, para que possamos aprimorar o aplicativo cada vez mais.
Quais dados estamos coletando e utilizando.

- Geolocalização
- Informações sobre o dispositivo e uso do aplicativo.
- Android advertising identifier

Porque a coleta de dados e como estamos utilizando seus dados.

- Geolocalização

Para entender melhor a localização dos usuários de nosso serviço.

- Informações sobre o dispositivo e uso do aplicativo(Coletados pelo Firebase)

Utilizamos estas informações para aprimorar nossos serviços e verificar os erros do aplicativo nos respectivos dispositivos para assim corrigi-los o quanto antes.

- Android advertising identifier

Utilizado pelo Firebase(firebase.google.com) e AdMob(admob.google.com) para veicular anúncios.


Os dados serão guardados por tempo indeterminado.

* Não são coletados dados referente a cadastro de usuários e informações pessoais.

* Reservamos o direito de modificar essa política de privacidade a qualquer momento, então por favor, revise-a com frequência.
