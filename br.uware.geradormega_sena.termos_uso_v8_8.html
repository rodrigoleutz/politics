Gerador de Números para Loteria(Termos de Uso)

br.uware.geradormega_sena


* Este aplicativo apenas gera números randômicos.
* Não é possível calcular o fechamento dos números quando gerados, apenas as sequencias não serem iguais, pois utiliza muito processamento, tornando inviável.
* Este aplicativo não faz as apostas dos números gerados.
* Este aplicativo não garante que os números gerados sejam os sorteados.
* Os resultados podem conter erros, verifique sempre em outras fontes também.
* As conferências de jogos estão em fase de testes e conferem apenas os números.
* Não são coletados dados referente a cadastro de usuários e informações pessoais.

* Reservamos o direito de modificar esses termos de uso a qualquer momento, então por favor, revise-a com frequência.
